# SponsorLens Demo Script (Updated)

## 60-second version
“International students often waste time on postings with unclear or conflicting sponsorship language. SponsorLens reads visible job text, scores work-authorization signals, flags contradictions, and returns Strong Fit / Risky Fit / Low Fit with confidence and evidence level. It also provides a recruiter message studio and lets users save analyses locally to track where they should focus.”

## 3-minute flow
1. Explain the problem: ambiguous or contradictory posting language.
2. Run Analyze on a page.
3. Highlight verdict, score, confidence, evidence level, and top evidence.
4. Show message studio (short/polite/cover variants).
5. Save analysis and show Saved Jobs summary (Strong/Risky/Low counts and total time saved).
6. Trigger demo scenarios for deterministic fallback.
7. Close with disclaimer: decision-support, not legal advice.
