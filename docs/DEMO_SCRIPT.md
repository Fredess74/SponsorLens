# Demo Script

## 30-Second Elevator Pitch

> "International students waste hours applying to jobs with hidden sponsorship restrictions. SponsorLens is a Chrome extension that reads what the posting actually says — not just what checkboxes the employer selected — and tells you whether to apply, clarify, or skip. Everything runs locally. No accounts, no API keys, no data collected."

## 3-Minute Walkthrough

### 1. The Problem (30s)
- Show a real job posting with ambiguous work-authorization language
- Point out how "must be authorized to work" could mean anything
- Note: students currently have no way to interpret this quickly

### 2. Live Analysis (60s)
- Click the SponsorLens icon → hit **Analyze this job**
- Walk through the result card:
  - **Verdict**: Strong Fit / Risky Fit / Low Fit with numeric score
  - **Confidence & Evidence**: How reliable is this analysis?
  - **Top Evidence**: Which specific phrases triggered the score
  - **Contradictions**: Any conflicting signals detected

### 3. Recruiter Message Studio (30s)
- Show the three message variants: Short / Polite / Cover Letter
- Click to switch between them
- Copy the selected message — ready to paste into an email or LinkedIn

### 4. Saved Job Tracker (30s)
- Save the analysis
- Show the summary: Strong/Risky/Low counts, total time saved
- Export as JSON for record-keeping

### 5. Demo Mode (15s)
- Click the three demo buttons to show deterministic Strong/Risky/Low scenarios
- Useful when no live job page is available

### 6. Close (15s)
- Recap: "All local. No API. No data collected. Saves students time and reduces blind applications."
- Disclaimer: decision-support tool, not legal advice
