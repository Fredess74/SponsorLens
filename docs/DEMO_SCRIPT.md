# SponsorLens Demo Script (Landing + Product)

## Opening note
Use the landing page as the opening and closing frame in the video for a polished narrative.

## 75–90 second outline
1. Student problem: international students face unclear sponsorship language.
2. Hidden restrictions: show examples like citizens-only, no sponsorship, clearance.
3. SponsorLens reveal: explain local-first extension workflow.
4. Strong / Risky / Low demo walkthrough.
5. Recruiter-safe message studio example.
6. Privacy-first MVP statement.
7. Closing frame: landing page + GitHub link + demo CTA.

## Short presenter line
“Job boards help students find jobs. SponsorLens helps international students decide which jobs are worth their time.”
