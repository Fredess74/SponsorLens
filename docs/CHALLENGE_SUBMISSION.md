## Submission strategy

## Suggested submission links
1. Landing page
2. GitHub repository
3. Demo video

## Project framing
SponsorLens is a local-first Chrome extension that helps international students evaluate job posting language for work-authorization friction, contradictions, and application risk before spending time on long applications.

## Why this is useful
- Reduces blind applications
- Surfaces hidden restrictions and contradictory statements
- Gives practical next-step messaging

## Privacy statement
- Local-first default
- No resume/document/ID collection
- No persistent full page text storage
- No external API calls required for MVP mode

## Disclaimer
Educational decision-support only. Not legal, immigration, or employment advice.
